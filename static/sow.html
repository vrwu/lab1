save our women - a women's safety app

navigate back

Inspired by recent events and increased media presence on women safety. This iOS app aims to promote the safety of women through awareness and action.

  • Coordinated Python Flask back end server with Google Firebase database and Twilio API SMS services
  • Utilized Google Maps and Places API for location tracking and coordinate transfer to database
  • Connected back end and React Native front end through Axios API, HTTP requests, and Expo

Key Features:

  • Emergency SOS Messaging
  • Location Tracking to Emergency Contacts
  • Creation of Incident Reports
  • Scenario Based Audios and Tips
